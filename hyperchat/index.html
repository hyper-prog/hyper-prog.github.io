HyperChat

Compact stand alone clinet/server graphical chat program (C# .NET/Mono)

HyperChat is a compact client/server chat program wihch allows to chat with graphical mode. The chat partners have a common drawing area so anyone can draw to it during the chat. The users can draw many ways an colors, and the pictures can be saved in file. At the start you can select the program network operation mode (Server/Client) Written in C# works and tested under windows .NET and Linux Mono (with Winforms lib)
