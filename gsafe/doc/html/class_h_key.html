Detailed Description

This is a field type class.
HKey holds a key data, which represent in the sql database. The value is readed from the database, but stays read only in gSAFE. Usally holds the primary key of the sql table. Many other function search this type to identify the records to read and write,
SO IT IS STRONGLY RECOMMEND THAT EVERY SQL TABLE SHOULD CONTAIN A PRIMARY KEY (Which should be a HKey here)

Gui appearance: The field have a simple readonly text label
